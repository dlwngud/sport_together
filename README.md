### 🛠️ 기술 스택

#### 개발 환경
- Android Studio
  - minSdk : 24
  - targetSdk : 34
  - jvmTarget : 1.8
#### 기술 스택
- Language: Kotlin
- Architecture : MVVM
- Library : Retrofit2, Glide, Lottie, Hilt, Room, Coroutine, Jetpack
- API : Kakao Login, Naver map API
